# 药品索引系统

一个简单易用的药品管理和查询系统，支持药师端管理和护士端查询。

## 功能特点

### 药师端（管理员）
- ✅ 上传药品名称和图片
- ✅ 查看所有药品列表
- ✅ 删除药品信息
- ✅ 实时数据同步

### 护士端（查询）
- ✅ 搜索药品
- ✅ 查看药品详情
- ✅ 实时查看最新数据

## 使用说明

1. 直接用浏览器打开 `index.html` 文件
2. 选择角色（药师端或护士端）
3. 输入对应的用户名和密码登录
4. 开始使用系统

## 数据存储与同步

系统使用 **GitHub 作为云端数据库**，实现跨设备数据同步：

- ✅ 药师端上传数据 → 自动保存到 GitHub
- ✅ 护士端查询 → 自动从 GitHub 读取最新数据
- ✅ 多设备实时同步（每30秒自动同步）
- ✅ 数据永久保存在 GitHub，不会丢失

### 配置步骤

详细配置说明请查看 `配置说明.md` 文件。

简要步骤：
1. 创建 GitHub Personal Access Token
2. 修改 `app.js` 中的 GitHub 配置
3. 上传文件到 GitHub 仓库
4. 开始使用

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- GitHub API（云端数据存储）
- LocalStorage（本地缓存）

## 浏览器兼容性

支持所有现代浏览器：
- Chrome
- Firefox
- Safari
- Edge

## 注意事项

- 图片会转换为 Base64 格式存储，建议上传小于 2MB 的图片
- 数据存储在 GitHub，需要配置 GitHub Token
- 系统每 30 秒自动同步一次数据
- 需要网络连接才能同步数据（本地缓存可离线查看）
